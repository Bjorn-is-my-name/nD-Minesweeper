# nD-Minesweeper
Minesweeper in every dimension

Im still working a lot of things out and the code is extremely messy but for now it works decent enough.

You wont need anything besides the nD-Minesweeper.exe file.
When running the game you can use the input fields to choose the dimension, their sizes and the amount of bombs.
E.g. Dimension 3 and all the sizes set to 5 will result in a 5 x 5 x 5

When everything is filled out you can press Start to start the game. There is no win or lose message yet, the game 
just stops and you can press Start again or change the game variables.

The game is finished once you either reveal a bombs (lose) or flag all bombs (win).
While playing you can press Stop at any time to stop the game.

Left mouse will reveal a cell
Right mouse will flag a cell

Pressing D will enable drawing mode
in this mode you can't draw yet, you can only move the board by holding down Right mouse

Its still full of little bugs so if you find one please report it to me by opening an Issue.
At the moment there is a bug in the sizes of 1st and 2nd dimension so i recommend setting those 2 to alteast size 3 to get no weird behaviour.
